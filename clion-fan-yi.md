# Clion翻译

```bash

代码处右键的选项:
    Copy Reference  <拷贝引用>
    Paste  <粘贴>
    Paste form History...  <粘贴板.曾经拷贝过的内容>
    Paste Simple  <粘贴简单>
    Column Selection Mode  <列选择模式>

    Find Usages  <查找用法>
    Refactor  <重构>{___
            Rename...  <全局重命名>
            Change Signature...  <更改制定函数内的变量名和类型>

            Move...  <重构移动>
            Copy...  <重构副本>
            Safe Delete...  <重构安全删除>

            Extract  <重构提取>
            Inline...  <重构内联>

            Pull Members Up...  <重构成员拉起>
            Push Members Down...  <重构推成员向下>
            }
    Folding  <折叠>{___
            Expand  <展开折叠>
            Collapse  <折叠>
            Expand Recursively  <递归式展开折叠>
            Collapse Recursively  <递归式折叠>
            Expand All  <展开所有折叠>
            Collapse All  <折叠所有>
            Expand to level  <展开折叠的等级>
            Expand all to level  <展开折叠的所有等级>
            Expand doc comments  <展开注释>
            Collapse doc comments  <折叠注释>
            Fold Selection / Remove region  <折叠或展开区域>
            Fold Code Block  <折叠代码块>
            }
    Go To  <去>{___
            jump to Navigation Bar  <跳转到导航栏>
            Declaration  <转到声明>
            Definition(s)  <转到定义>
            Type Declaration  <转到类型声明>
            Super Definition  <转到超级定义>
            }
    Generate...  <生成>

    Reveal in Finder  <在访达Finder中显示>
    Open in terminal  <在终端打开>

    Local History  <地方历史>{___
            Show history  <显示修改历史>
            Show History for Selection  <显示选择区域的历史>
            Put Label...  <放置一个贴士,用来记录一些东西>
            }
    Compare with Clipboard  <与剪切版比较>
    File Encoding  <文件编码>

    Create Gist...  <创建要点>



@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@

键盘代码区操作 ⌃ + ENTER   (左下角按键 + 回车键)
    Constructor  <构造函数>
    Destructor  <析构函数>
    Getter  <消气>
    Setter  <设定器>
    Getter and Setter   <吸气剂和定位器>
    Equality Operators  <相等算子>
    Relational Operators  <关系运算符>
    Stream Output Operator  <流输出运算符>
    Override Functions...  <覆盖功能...>
    Implement Functions...  <实现功能...>
    Generate Definitions...  <生成定义...>

```
